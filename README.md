Requirements:
- Java 17 or later
- A webcam connected to your computer

How to Run:

Windows:
Double-click `run.bat`, or run it from the terminal:
run.bat


Manual:
From inside the `AttendanceSystem` folder, run:
MainApp.java from AttendanceSystem\src\main\java\attendance\MainApp.java

Notes:
Pre-compiled class files are in the `out/` folder — no build step needed.
JavaFX libraries are bundled in the `javafx/` folder.
OpenCV native libraries are in the `libs/` folder.
Face data and Haar cascade XML files are in the `data/` folder.
